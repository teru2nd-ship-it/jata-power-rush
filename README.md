# 猫試し / JATA POWER RUSH

`猫騙し`の語感をもじった日本語タイトル「猫試し」。数字を吸収して成長し、猫侍が師範へ挑むスマートフォン向けHTML Canvasゲームです。英語タイトルと公開URLは従来どおりです。

## 公開先

- Game: https://jata-power-rush.vercel.app/
- GitHub: https://github.com/teru2nd-ship-it/jata-power-rush

## 遊び方

- 自分より小さい数字の竹を選ぶと、その数字を吸収します。
- 忍者とクノイチの「加算の術 / 増幅の術」を選び、マタタビの×2バフでパワーを伸ばします。
- 自分より小さい師範を倒すと次の修行へ進みます。
- タイトルの「魚釣り編」では、同じ数字吸収ルールで川から磯、沖へ続く長い一本道を進みます。
- 魚・貝・カニ・ロブスターなどを捕まえて釣果を伸ばし、最後の巨大マグロを狙います。
- 巨大マグロを確保すると、魚釣り編専用のエンディング映像が再生されます。
- 30の修行を終えるとエンドカードが表示され、タイトルへ戻ります。
- 1面から30面まで一度もゲームオーバーにならず達成すると、ノーミス特典映像が開放されます。

タイトル画面では、黒猫の女侍と茶トラの男侍を選択できます。
和風チップチューンBGMと効果音は右上の音声ボタンで切り替えられます。上端の共通メニューからゲーム一覧・全画面表示を利用できます。

## ローカル起動

```sh
python3 -m http.server 8787 --bind 0.0.0.0
```

ブラウザーで `http://localhost:8787/` を開きます。

## 構成

- `index.html`: ゲーム本体
- `assets/arcade-game-menu.js`: JATA ARCADE共通メニュー・音声・共有ボタン
- `assets/og-neko-dameshi.png`: 公開共有・JATA ARCADEカード用1200×630画像
- `assets/ending/`: 30面エンドカード・ノーミス特典映像・魚釣り編クリア映像
- `assets/backgrounds/`: 道場・日本城、魚釣り編の川・磯・沖背景
- `assets/enemies/`: 竹、忍者、クノイチ、ボス、魚釣り編の魚介7種・巨大マグロ
- `assets/heroes/`: 黒猫侍・茶トラ侍、釣り竿と網を装備した魚釣り編版

画像素材の元データは各`source/`フォルダに保存しています。
魚釣り編は透過済みの魚介7種と、波付きの巨大マグロ専用スプライトを使用します。
